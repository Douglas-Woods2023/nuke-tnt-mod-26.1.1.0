# Nuke TNT
This is a Nuke TNT Mod.
You can use a nuclear bomb to destroy any buildings you have constructed and your computer.

Module Runtime Environment:
============
1. Minecraft Java Edition 26.1.1
2. NeoForge >= 26.1.1.0-beta

Instructions for use:
============
1. Combine 9 spider eyes to make a block of uranium.
2. Craft a nuclear bomb with 4 obsidian, 3 iron ingots, 1 uranium block, and 1 TNT.
3. Sneak in and use a nuclear bomb to detonate it.
